# simple-word-battle
这是一个用于学习英语单词的趣味单词PK小游戏。
依赖于Node运行后端API服务。
用户可以选择单人模式或多人模式，每轮将随机抽选一个单词和四个选项，并在用户选择一个选项后给予判断反馈，并自动刷新一轮题目。
在双人模式下，双方独立计数累积正确数，可以设置全局倒计时，增加趣味。
该项目开发的目的是促进学生趣味学习英语单词，它是在我高中时期完成的作品，现在看来质量有些不堪，但也的确给我们班级的同学带来过一些惊讶和趣味。
该项目的设计并不规范，因此您下载之后可能需要做出一些调整。这需要您对Node和Web有一定理解能力和基础知识。
该项目已不再计划继续更新迭代，仅作为纪念展示。如果您有任何想法，欢迎联系我。
